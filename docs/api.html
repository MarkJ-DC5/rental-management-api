Retrieve rooms of a specific building.

Supports pagination, sorting (e.g., by roomName, rent), and filtering by room status.

path Parameters
buildingId
required
integer <int32>
query Parameters
page
integer >= 0
Default: 0

Zero-based page index (0..N)

size
integer >= 1
Default: 20

The size of the page to be returned

sort
Array of strings
Default: "roomName,ASC"

Sorting criteria in the format: property,(asc|desc). Default sort order is ascending. Multiple sort criteria are supported.

status
string
Default: "all"
Enum: "VACANT" "OCCUPIED" "ALL"

Responses

Local

http://localhost:8080/api/v1/buildings/{buildingId}/rooms

Retrieve tenants of a specific building.

Supports pagination, sorting (e.g., by dateMovedIn, firstName), and filtering by tenant status (active = currently living, inactive = moved out, all) and by whether the tenant is the primary tenant of a room.

path Parameters
buildingId
required
integer <int32>
query Parameters
page
integer >= 0
Default: 0

Zero-based page index (0..N)

size
integer >= 1
Default: 20

The size of the page to be returned

sort
Array of strings
Default: "room.roomId,ASC"

Sorting criteria in the format: property,(asc|desc). Default sort order is ascending. Multiple sort criteria are supported.

status
string
Default: "all"
Enum: "ACTIVE" "INACTIVE" "ALL"
primaryOnly
boolean
Default: true

Responses

Local

http://localhost:8080/api/v1/buildings/{buildingId}/tenants
